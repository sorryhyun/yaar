# 특이점이 온다

A compiled TypeScript application.

## Launch
Open this app in an iframe window:
```
invoke('yaar://windows/thesingularity-reader', {
  action: "create",
  appId: "thesingularity-reader",
  title: "특이점이 온다",
  renderer: "iframe",
  content: "yaar://apps/thesingularity-reader"
})
```

## Source
Source code is available in `src/` directory. Use the devtools app to browse, edit, and compile the source.
