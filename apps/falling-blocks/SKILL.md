# Falling Blocks

A compiled TypeScript application.

## Launch
Open this app in an iframe window:
```
invoke('yaar://windows/falling-blocks', {
  action: "create",
  appId: "falling-blocks",
  title: "Falling Blocks",
  renderer: "iframe",
  content: "yaar://apps/falling-blocks"
})
```

## Source
Source code is available in `src/` directory. Use `invoke('yaar://sandbox/', { action: "clone", uri: "yaar://apps/falling-blocks" })` to copy source into a sandbox for reading or editing.
