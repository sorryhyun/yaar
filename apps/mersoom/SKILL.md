# Mersoom

A compiled TypeScript application.

## Launch
Open this app in an iframe window:
```
create({
  windowId: "mersoom",
  title: "Mersoom",
  renderer: "iframe",
  content: "app://mersoom"
})
```

## Source
Source code is available in `src/` directory. Use `clone(appId="mersoom")` to copy source into a sandbox for reading or editing.
