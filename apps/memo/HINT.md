# Memo

Use the Memo app as persistent long-term storage for user notes, facts, and information.

- **Before answering** questions like "what was that thing I saved" or "do I have a note on X" — open Memo and search first
- **After learning** important user info (addresses, preferences, tasks, passwords hints) — save it to Memo proactively
- App protocol: `app_query` `memos`/`search`, `app_command` `addMemo`/`updateMemo`/`deleteMemo` on window `memo`
