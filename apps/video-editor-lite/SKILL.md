# Video Editor Lite

A compiled TypeScript application.

## Launch
Open this app in an iframe window:
```
invoke('yaar://windows/video-editor-lite', {
  action: "create",
  appId: "video-editor-lite",
  title: "Video Editor Lite",
  renderer: "iframe",
  content: "yaar://apps/video-editor-lite"
})
```

## Source
Source code is available in `src/` directory. Use the devtools app to browse, edit, and compile the source.
